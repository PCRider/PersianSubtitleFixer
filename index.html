Qt5 Application

Some Features:
  • You can drag file to the Window
  • Open and Save File with File dialogs
  • There is an button to change text font

For compile please run codes below:

mkdir build 

cd build 

qmake .. 

make

Then, After that, you can use persiansubtitlefixer in build folder: 

 chmod +x ./persiansubtitlefixer 

sudo cp persiansubtitlefixer /usr/bin/PersianSubtitleFixer 

sudo chmod +x /usr/bin/PersianSubtitleFixer

After this commands, You can run Application with codes in below: 

 PersianSubtitleFixer
